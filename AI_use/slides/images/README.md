# Images Directory

Place your images here and reference them in your slides using:

```markdown
![Image description](images/your-image.png)
```

## Recommended Image Formats
- PNG for diagrams and screenshots
- JPG for photographs
- SVG for vector graphics (scalable)
- GIF for simple animations

## Tips
- Keep file sizes reasonable (< 1MB per image)
- Use descriptive filenames
- Optimize images before adding to repository
